# Responsive Search Ads

## Status

`Beta`

## Overview

Responsive search ads (RSAs) dynamically assemble headlines and descriptions to match user search queries.
Google Ads tests combinations of provided assets and learns which combinations perform best over time.

Recommendation:

- Maintain at least `2` RSAs per ad group.
- Target `Good` or `Excellent` Ad Strength.
- Use a unique final URL for each RSA.

Related resources:

- Ad Strength: https://support.google.com/google-ads/answer/9921843
- Search Automation technical guide: https://goo.gle/unlockthepowerofsearch
- Creating effective Search ads: https://support.google.com/google-ads/answer/6167122

## Best Practices

### Asset Requirements

- Assets may appear in any order.
- Each asset must make sense independently and in combination with other assets.
- Assets must comply with Google Ads policies and applicable laws.

### Ad Strength Recommendations

- Implement at least `2` RSAs per ad group with `Good` or `Excellent` Ad Strength.
- Use a unique final URL for each RSA.

Performance observations:

- Improving Ad Strength from `Poor` to `Excellent` results in an average `15%` increase in clicks and
  conversions.¹
- Adding a second RSA to an ad group increases conversions by an average `6.6%` at a similar cost per
  conversion.¹
- Adding a third RSA to an ad group increases conversions by an average `3.7%` at a similar cost per
  conversion.¹

### Required Content

If specific text must appear in every ad, pin it to:

- `Headline position 1`
- `Headline position 2`
- `Description position 1`

## Benefits

Responsive search ads provide:

- Flexible ad layouts that adapt to device widths.
- Automated testing of headline and description combinations.
- Location-based message customization.
- Increased auction eligibility through additional asset combinations.
- Expanded query matching opportunities.
- Potential improvements in clicks and conversions.

## How It Works

### Asset Combination Process

Provide multiple headlines and descriptions.

Google Ads:

1. Generates multiple ad combinations.
2. Avoids redundant combinations.
3. Tests combinations across search queries.
4. Learns which combinations perform best.

RSA limits:

| Asset Type   | Minimum | Maximum | Character Limit    |
| ------------ | ------- | ------- | ------------------ |
| Headlines    | 3       | 15      | 30 characters each |
| Descriptions | 2       | 4       | 90 characters each |
| Paths        | N/A     | N/A     | 15 characters each |

Notes:

- At least one headline and one description are shown in each ad.
- Matching query text may appear in bold.
- In double-width languages such as Korean, Japanese, and Chinese, each character counts as `2` characters.

Reference:

- Text ad requirements: https://support.google.com/adspolicy/answer/6021630

## Enhanced Flexibility

RSAs can display assets beyond traditional placements when predicted to improve performance.

### Within the Same Ad

Headline assets may:

- Appear at the beginning of a description.
- Serve as link-based assets.

Additional behavior:

- Up to `2` unused RSA headlines may serve as link-based assets.
- RSA descriptions may populate link-based asset description space.
- Link-based assets point to the same domain as the ad's final URL.

### Within the Same Ad Group

Unused RSA headlines and descriptions from another RSA in the same ad group may serve as link-based assets.

Behavior:

- Borrowed assets point to the final URL of the source RSA.

### Restrictions

- Only active ads and active assets are eligible.
- Sensitive verticals are not eligible for asset sharing.

### Interaction with Pinned Assets

Assets pinned to:

- `Headline position 1`
- `Headline position 2`
- `Description position 1`

remain in their assigned positions.

Reference:

- Pinning assets: https://support.google.com/google-ads/answer/7684791#positionpinning

## Create a Responsive Search Ad

### Procedure

1. Open Google Ads.
2. Select `Campaigns`.
3. Select `Ads`:
   - https://ads.google.com/aw/ads

4. Select `+`.
5. Select `Responsive search ad`.
6. Enter:
   - Final URL
   - Display path text

7. Enter headlines:
   - Minimum: `3`
   - Maximum: `15`

8. Enter descriptions:
   - Minimum: `2`
   - Maximum: `4`

9. Select `Save`.

### Preview Behavior

The preview:

- Displays possible asset combinations.
- Does not display all combinations.
- May shorten content in some formats.

### Headline Optimization Guidelines

- Assume the first three headlines may appear together.
- Include at least one target keyword in headlines.
- Ensure relevance to targeted keywords.
- Create as many unique headlines as possible.
- Highlight:
  - Product benefits
  - Service benefits
  - Problems solved
  - Shipping information
  - Return policies

- Create headline variations with different calls to action.

Reference:

- Creating effective Search ads: https://support.google.com/google-ads/answer/6167122

### Campaign Creation Notifications

During campaign creation, Google Ads may display notifications regarding:

- Targeting
- Bidding
- Budget
- Other campaign settings

Notifications may indicate:

- Reduced performance risk
- Configuration issues that prevent publishing

Reference:

- Set up your campaign for success: https://support.google.com/google-ads/answer/9451609

## Edit a Responsive Search Ad

### Procedure

1. Open Google Ads.
2. Select `Campaigns`.
3. Select `Ads`:
   - https://ads.google.com/aw/ads

4. Add a filter.
5. Select:
   - `Attributes`
   - `Ad type`

6. Select `Responsive search ad`.
7. Hover over the target ad.
8. Select the edit icon.
9. Select `Edit`.
10. Modify:
    - URL
    - Final URL
    - Headlines
    - Descriptions
    - URL options

11. Select `Save`.

### Important

Editing or removing headlines or descriptions may:

- Change serving combinations.
- Affect performance.

## Pin Headlines and Descriptions to Specific Positions

### Purpose

Pinning controls where specific assets appear.

Example:

To ensure a disclaimer appears in every ad:

1. Add the disclaimer as a description.
2. Pin it to `Description position 1`.

### Procedure

1. Create or edit an RSA.
2. Select the pin icon beside a headline or description.
3. Choose a position.
4. Repeat as needed.
5. Select `Save`.

### Important

Editing or removing pinned assets may affect serving behavior and performance.

### Pinning Guidelines

- Pinning restricts asset placement.
- Reduced flexibility can lower Ad Strength.
- Pinning is not recommended unless placement control is required.

Recommended approach:

- Pin `2–3` assets per position when possible.
- Allow multiple assets to compete within the same position.

Example:

If both of the following are pinned to `Headline position 1`:

- `Official Website`
- `The Official Site`

then either headline may appear first.

### Pinning Rules

- Assets pinned to all available positions prevent unpinned assets from serving.

- Assets pinned to:
  - `Headline position 1`
  - `Headline position 2`
  - `Description position 1`

  always appear.

- Assets pinned to:
  - `Headline position 3`
  - `Description position 2`

  are not guaranteed to appear.

If content must appear in every ad, pin it to:

- `Headline position 1`
- `Headline position 2`
- `Description position 1`

## Reporting on Responsive Search Ad Performance

### Overview

The `Ads` page provides the same performance metrics available for other text ads.

Statistics represent aggregated performance across all combinations generated from the RSA's assets.

### Generate a Report

1. Open Google Ads.
2. Select `Campaigns`.
3. Select `Ads`:
   - https://ads.google.com/aw/ads

4. Select:
   - `Attributes`
   - `Ad type`

5. Select `Responsive search ad`.
6. Optional: Modify report columns.
   - Select the columns icon.
   - Select `Modify columns`.
   - Choose desired columns.
   - Reference: https://support.google.com/google-ads/answer/2454071

7. Select `Apply`.
8. Select the download icon.
9. Choose an export format:
   - `.csv`
   - `.pdf`

Optional actions:

- Open in Report Editor.
- Schedule report generation.
- Email reports.

References:

- Report Editor: https://support.google.com/google-ads/answer/7489070
- Create, save, and schedule reports: https://support.google.com/google-ads/answer/2404176
